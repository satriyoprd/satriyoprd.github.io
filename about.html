Arti Logo Hima Energi

Lambang merupakan simbol identitas yang mewakili HIMA ENERGI PENS. Berbentuk vektor api yang saling terangkai membentuk huruf “E” Vektor api berwarna coklat yang ke arah kanan berjumlah tiga dan vektor api berwarna hijau ke arah kiri.Di bawah lambang himpunan terdapat tulisan “HIMA ENERGI” dengan font brave.



Vector Api

Vektor api memiliki arti semangat.Warna vektor api kombinasi warna coklat dan hijau dimana warna coklat memiliki arti netral (bijaksana) dan warna hijau memiliki arti natural.

Font & Warna

Font Brave dipilih dengan maksud sebagai asosiasi sifat berani. Coklat berasal dari warna tanah, merupakan implementasi dari impartial yakni tidak memihak
